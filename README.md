# Student Management System (Django)

## 📌 Overview
This is a web-based Student Management System built using Django. It allows users to manage student records efficiently with CRUD operations.

## 🚀 Features
- Add new student records
- Update existing student details
- Delete student records
- View student list
- Search functionality

## 🛠️ Tech Stack
- Python
- Django
- SQLite

## ⚙️ Installation & Setup
1. Clone the repository:
   git clone https://github.com/anam0015/SMS_project

2. Navigate to project folder:
   cd SMS_project

3. Install dependencies:
   pip install -r requirements.txt

4. Run the server:
   python manage.py runserver

5. Open in browser:
   http://127.0.0.1:8000/

## 📌- User Signup (Register new users)
- User Login & Logout
- Authentication-based access control
- Only logged-in users can manage student data
